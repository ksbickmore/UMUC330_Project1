Under Libraries add stdc++fs

MINGW32

There are bugs in the MINGW32 compiler that mishandle c++11 functionality such as std::stoi or std::to_string.

MINGW64

There are bugs in the MINGW64 installer that did not allow me to ever properly install. Source code installation
failed do to missing libraries.

Cygwin

The Cygwin g++ compiler fails to properly resolve templates.
for instance passing a string into the path constructor does not resolve properly.
template< class Source >
path( const Source& source );
