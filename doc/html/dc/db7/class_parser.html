Detailed Description

The Parser class is used to parse each Token created by the Lexer class and to parse the overall syntax of the grammar:
gui ::= Window STRING '(' NUMBER ',' NUMBER ')' layout widgets End '.'
layout ::= Layout layout_type ':'
layout_type ::=
Flow '(' [ align ] ')'|
Border '(' [ NUMBER ',' NUMBER ] ')'|
Grid '(' NUMBER ',' NUMBER [',' NUMBER ',' NUMBER] ')'
align ::= LEFT | RIGHT | CENTER
widgets ::= widget widgets | widget
widget ::=
Button STRING ';' |
Group radio_buttons End ';' |
Label STRING ';' |
Panel layout widgets End ';' |
Textfield NUMBER ';'
radio_buttons ::= radio_button radio_buttons | radio_button
radio_button ::= Radio STRING ';'
